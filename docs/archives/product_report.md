# EnlightenLM 产品报告

## 1. 项目概述

EnlightenLM 是一个基于认知神经科学的大模型安全推理与元认知框架，旨在为大语言模型提供实时自我监控与安全截断能力。

**项目定位**：
- 安全增强型大语言模型框架
- 基于认知神经科学原理
- 三层架构设计
- 支持 DeepSeek API 集成

**应用场景**：
- 智能对话系统
- 安全内容生成
- 教育辅助工具
- 企业级 AI 应用

## 2. 核心架构

### 2.1 三层架构设计

| 层级 | 功能 | 实现文件 | 技术特点 |
|------|------|----------|----------|
| L1 生成层 | 文本生成 | `enlighten/l1_generation.py` | 基于 DeepSeek API 的高效生成 |
| L2 工作记忆层 | 上下文管理 | `enlighten/l2_working_memory.py` | 注意力机制与上下文追踪 |
| L3 元控制器 | 安全监控 | `enlighten/l3_controller.py` | 实时安全评估与截断 |

## 5. 系统部署

### 5.1 部署要求

| 环境 | 要求 | 备注 |
|------|------|------|
| 操作系统 | Windows/Linux/Mac | 支持所有主流系统 |
| Python | 3.8+ | 核心运行环境 |
| 网络 | 互联网连接 | 用于 API 调用 |
| 硬件 | 2GB+ 内存 | 基本运行需求 |

### 5.2 部署步骤

1. **克隆项目**
```bash
git clone https://github.com/your-org/enlightenlm.git
cd enlightenlm
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **配置环境**
```bash
# 设置 API Key
export DEEPSEEK_API_KEY="your-api-key"

# 启动服务
python -m enlighten.api_server
```

4. **启动前端**
```bash
cd docs
python -m http.server 8080
```

### 5.3 访问地址

| 服务 | 地址 | 说明 |
|------|------|------|
| 聊天界面 | http://localhost:8080/chat.html | Web UI |
| API 服务 | http://localhost:8000 | REST API |
| API 文档 | http://localhost:8000/docs | Swagger 文档 |
